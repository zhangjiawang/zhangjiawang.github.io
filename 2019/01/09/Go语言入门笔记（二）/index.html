Go语言入门笔记（二）

发表于 2019-01-09

运算符

  1. 算数运算符：+ - * / % ++ --
  2. 关系运算符：== != > < >= <=
  3. 逻辑运算符：&& || !
  4. 位运算符：
    • & 按位与：都为 1 则结果为 1
    • | 按位或：有一个为 1 则结果为 1
    • ^ 按位异或：对应位不同值则结果为 1
    • << 左移：把数整体向左移
    • >> 右移：把数整体向右移
    • &^ 位清空：被运算变量 bit 位上数字为 0 ，则取运算变量对应位置上的数字，如果被运算变量 bit 位上数字为 1 ，则结果位上取 0
  1. 赋值运算符：= += -= *= /+ %= &= |= ^= >>= <<=

控制语句

  1. 条件语句：if...else，嵌套 if...else if...
  2. 选择语句：switch，select
  3. 循环语句：for
  4. 关键字：goto，break，continue

函数

  1. 函数定义格式

    func 函数由 func 开始声明
    function_name 函数名称
    parameter list 参数列表，指定参数类型、顺序、个数，可选
    return_type 返回值类型，可选

    函数可返回多个值
